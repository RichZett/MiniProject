Detailed Description

Button input control for the system.

This task reads the button control input of three buttons, and changes the state of the system accordingly. It can toggle ON/OFF the system and increase/decrease the temperature setpoint.

The execution time of each read cycle is measured using k_uptime_get() and the longest duration observed is tracked.

The task monitors the button input and updates state of the system.

This task runs continuously, reads the input from three buttons as follows:

  • Button 0 is used to toggle the system ON/ OFF.
  • Button 1 is used to increase the temperature setpoint.
  • Button 2 is used to decrease the temperature setpoint.
